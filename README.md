# Bank Customer Churn Prediction

This project focuses on predicting customer churn in retail banking using machine learning techniques.

## Project Overview
The goal of this project was to analyze which factors influence customer churn and build predictive models that can classify whether a customer is likely to leave the bank.

## Main Work Done
- Performed exploratory data analysis (EDA)
- Cleaned and preprocessed the dataset
- Selected relevant features
- Trained and compared multiple machine learning models:
  - Logistic Regression
  - Random Forest
  - SVM
  - Gradient Boosting

## Result
Among the evaluated models, Gradient Boosting achieved the best overall performance.
